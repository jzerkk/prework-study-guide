HTML

Markdown Language Cheat Sheet :

  • Element: A type of content on a webpage -
  • Head Element - Contains information (metadata) that helps the browser render the page correctly. Not visible when the user looks at the webpage.
  • Body Element - Contains the content that is shown to the user. All of the elements we would like the user to see and interact with must be placed inside of a Body Element
  • Text elements - Contains text within elements such as paragraphs and subheadings.
  • Image elements - "img" and uses a src attribute to retrieve it's information from.
  • List elements - This is a list element
  • Meta-related elements - Such as link (Css file link), UTF-8, Name = "viewport" and content="Width=device-width, initial-scale=1.0",
  • Content division elements - Divides page, acts as a good border too.
  • Semantic elements - The name of the element discribes it's content.
  • Creates the sturdy “Frame” of the site.

JavaScript

JavaScript Cheat Sheet :

  • Function - A set of instructions that tells the computer how to perform a certain task. Functions do not automatically execute when the JavaScript file is run. You must call a function by name in order to execute it. Function name is always followed by parentheses.
  • Variables - Containers that store data (values). You start by declaring a variable with the “let” keyword, followed by the name you give the variable. Statement ends with a semi-colon if others are in line with it.
  • Array - This is a structure that allows you to store multiple values in a single reference. They are Zero Sequenced and Sequential.
  • String - This is a sequence of text known as a string. To signify that the value is a string, enclose it in single quote marks.
